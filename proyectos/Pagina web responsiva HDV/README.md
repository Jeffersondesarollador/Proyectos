# Hoja de Vida Web Responsiva

Proyecto de una hoja de vida digital desarrollada como página web estática, enfocada en una presentación clara de la información personal y profesional.

## Tecnologías utilizadas
- HTML5
- CSS3

## Características
- Diseño 100% responsivo
- Estructura semántica
- Adaptable a diferentes resoluciones de pantalla
- Proyecto liviano y rápido de cargar
